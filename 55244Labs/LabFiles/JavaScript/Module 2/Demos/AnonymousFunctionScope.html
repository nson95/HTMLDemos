Anonymous Functions and Scope

To see the global functions list, open the Developer Tools, click the Console tab and type “debugger;” press Enter and then find and open “Scopes”.
  • Listed functions ended up in the Global scope
    • Functions DoWork1, DoWork2, DoWork3, DoWork4 and DoWork4b will be seen from all code in this page.
    • Function DoWork3b will be seen from all code in this page, but is not attached to the window object.
  • Functions not listed there are only available in the scopes they were created in.
    • Functions DoWork2b and DoWork2c will only be accessible from code in function DoWork2.
    • Function DoWork4c will only be accessible in function DoWork4.
